# Acelerador Tests DGT #
Acelera tu aprendizaje con los tests de la DGT con accesos directos para las
opciones. Aprieta una tecla (`J`, `K`, `L`) de tu teclado para seleccionar la
opción que quieres y pasar a la siguiente pregunta.

![Demo Acelerador Tests
DGT](https://github.com/juanpescador/AceleradorTestsDGT/blob/master/demo-captions.gif)

## Teclas ##
Las teclas `J`, `K` y `L` se han escogido porque están juntas y, de
izquierda a derecha, representan la primera, segunda y tercera opción de
respuesta.

También puedes usar las teclas `A`, `B` y `C`, pero solo marcan la opción
correspondiente. Para pasar de pregunta utiliza las flechas izquierda y derecha.

## Instalación ##
Necesitas tener Chrome, con la extensión
[Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
instalada ([sitio oficial](https://tampermonkey.net/)).

Una vez instalada la extensión:

1. Navega hasta `src/testsdgtdirecto.user.js` y pincha en el botón `Raw`
([enlace
directo](https://github.com/juanpescador/AceleradorTestsDGT/raw/master/src/testsdgtdirecto.user.js)).
2. Tampermonkey te mostrará los detalles del script. Pincha en el botón
`Install`.  3. Escoge un [test de la
DGT](https://sedeapl.dgt.gob.es/WEB_EXAM_AUTO/service/TiposExamenesServlet).
